评审维度：1) 品牌人格（家庭、温暖、信任）；2) 与 magenta 的色彩呼应；3) 24px logo 旁的视觉权重平衡；4) 不喧宾夺主，因为 sidebar 还要承担 nav；5) collapsed 视图下 logo mark 单独存在，wordmark 不出现，所以字体只在 expanded 视图发挥。

下一步：哥选定方向后，会动 src/app/layout.tsx 引入字体 + src/components/AppSidebar.tsx 改 className；新增字体会增加首屏 web font 体积约 30-50 KB（subset latin），需要确认是否接受。

不做的事：除非哥确定方向，否则 src/ 一行不动。本文档单纯用于评审。
